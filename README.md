基本NLP工具
1、pyltp
https://github.com/HuangFJ/pyltp
提供分词、词性标注、命名实体识别、依存句法分析、语义角色标注等功能

2、fnlp
https://github.com/FudanNLP/fnlp
可实现以下功能：
    信息检索： 文本分类 新闻聚类
    中文处理： 中文分词 词性标注 实体名识别 关键词抽取 依存句法分析 时间短语识别
    结构化学习： 在线学习 层次分类 聚类
